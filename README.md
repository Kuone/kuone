# kuone

`kuone.github.io` linked to `kuone.io`, welcome to visit it ( [kuone.io](http://www.kuone.io) ).

## Installation and Run

Execute:

    $ bundle

Or install it yourself as:

    $ gem install material-theme

And then execute:
	
	$ jekyll serve
